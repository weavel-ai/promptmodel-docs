# DevClient

**DevClient** is used to register Promptmodel SDK usage and include it to _DevApp_.

## Methods

### register

A decorator to register Promptmodel usage inside a function.

```python filename="your_code.py"
@client.register
def my_function():
    ...
    outputs = PromptModel("my_prompt").run_and_parse(inputs=INPUTS)
	response_message = ChatModel("my_chat").run()
    # my_prompt and my_chat will be displayed in the development dashboard
    ...
```
