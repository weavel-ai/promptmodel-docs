# DevClient

**DevClient** checks your Promptmodel SDK usage and send it to the development dashboard.

## Methods

### register

A decorator to register Promptmodel instance usage inside a function to the development dashboard.

```python filename="your_code.py"
@client.register
def my_function():
    ...
    outputs = PromptModel("my_prompt").run_and_parse(inputs=INPUTS)
	response_message = ChatModel("my_chat").run()
    # my_prompt and my_chat will be displayed in the development dashboard
    ...
```
